WP-tagcloud displays jquery based animated tag cloud which is fully seo friendly.

== Description ==

wp-tag-cloud plugin create and displays the jquery based tag cloud. It is fully javascript and seo friendly and is not dependent on flash. The user can control the witdth,height,background color and text color.
The canvas element is not supporting on some browsers specially in ie although there are some hacks to run canvas element on them but it is very slow so i didnt used them. So on the non canvas supporting browser it will display text based tag cloud.

== Installation ==

= Installation =
1. Make sure you're running WordPress version 2.3 or better. It won't work with older versions.
2. Download the zip file and extract the contents.
3. Upload the 'wp-tag-cloud' folder to your plugins directory (wp-content/plugins/).
4. Activate the plugin through the 'plugins' page in WP.
5. See 'settings->Cloud Tag' for settings...
6. In order to actually display the tag cloud, you can put display_cloud_tag() anywhere you want to display the tag cloud.
7. Please fill all of the fields in the admin's settings page to display the plugin properly.
8. Please read the readme file included in the plugin directory for more help.
9. Demo: http://jargon-solutions.com/demos/tagcloud/
